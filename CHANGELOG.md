# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2026-05-27

### Added

- Initial release of `restic-profile` Python CLI and Ansible roles.
- Profile-based restic wrapper supporting custom backup schedules, hooks, and environment configurations.
- Ansible roles for deploying `restic_profile` client and `restic_rest_server`.
- Full unit test coverage and type validation checks.
